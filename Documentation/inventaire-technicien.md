# Documentation - inventaire/technicien.html

## Vue d'ensemble

Page de redirection vers `collaborateurs.html`. Cette page redirige automatiquement l'utilisateur vers la page des collaborateurs.

## Fonctionnalités

- **Redirection automatique** : Redirection immédiate via meta refresh et JavaScript
- **Compatibilité** : Double méthode de redirection pour assurer la compatibilité

## Sécurité

- ✅ Pas de traitement de données sensibles
- ✅ Redirection simple sans exposition de données

## Scripts et dépendances

- Aucun script externe nécessaire (redirection immédiate)

## Notes de développement

- Cette page sert uniquement de redirection
- La page réelle à documenter est `collaborateurs.html`
- La redirection est faite via `<meta http-equiv="refresh">` et `window.location.href`
